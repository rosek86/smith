Admittance to Reflection Coefficient

`\Gamma = (Z_L - Z_0) / (Z_L + Z_0), Z = 1 / Y`

`\Gamma = (1 / Y_L - 1 / Y_0) / (1 / Y_L + 1 / Y_0) = ((Y_0 - Y_L) / (Y_L * Y_0)) / ((Y_0 + Y_L) / (Y_L * Y_0)) = (Y_0 - Y_L) / (Y_0 + Y_L)`

Normalize

`\Gamma = (Y_0/Y_0 - Y_L/Y_0) / (Y_0/Y_0 + Y_L/Y_0) = (1 - Y_L/Y_0) / (1 + Y_L/Y_0)`

`y_L = Y_L/Y_0`

`\Gamma = (1 - y_L) / (1 + y_L)`


`y_L = y_R + j y_I`


`\Gamma = (1 - (y_R + j y_I)) / (1 + y_R + j y_I)`

`\Gamma = (1 - (y_R + j y_I)) / (1 + y_R + j y_I) (1 + y_R - j y_I) / (1 + y_R - j y_I)`

`\Gamma = (1 - y_R - j y_I) / (1 + y_R + j y_I) (1 + y_R - j y_I) / (1 + y_R - j y_I)`

`\Gamma = (1 + y_R - j y_I - y_R - y_R^2 + j y_R y_I - j y_I - j y_R y_I + j^2 y_I^2) / (1 + y_R - j y_I + y_R + y_R^2 - j y_R y_I + j y_I + j y_R y_I - j^2 y_I^2)`

`\Gamma = (1 - j y_I - y_R^2 - j y_I + j^2 y_I^2) / (1 + y_R + y_R + y_R^2 - j^2 y_I^2)`

`\Gamma = (1 - y_R^2 - y_I^2 - j 2 y_I) / ((1 + 2 y_R + y_R^2) + y_I^2)`

`\Gamma = (1 - y_R^2 - y_I^2 - j 2 y_I) / ((1 + y_R)^2 + y_I^2)`

`\Gamma = (1 - y_R^2 - y_I^2) / ((1 + y_R)^2 + y_I^2) + j (-2 y_I) / ((1 + y_R)^2 + y_I^2)`


Reflection Coefficient to Admittance

`\Gamma = (1 - y_L) / (1 + y_L)`

`(1 + y_L) \Gamma = 1 - y_L`

`\Gamma + y_L \Gamma = 1 - y_L`

`y_L + y_L \Gamma= 1 - \Gamma`

`y_L (1 + \Gamma) = 1 - \Gamma`

`y_L = (1 - \Gamma) / (1 + \Gamma)`


`y_L = (1 - \Gamma_R^2 - \Gamma_I^2) / ((1 + \Gamma_R)^2 + \Gamma_I^2) + j (-2 \Gamma_I) / ((1 + \Gamma_R)^2 + \Gamma_I^2)`


`g = (1 - \Gamma_R^2 - \Gamma_I^2) / ((1 + \Gamma_R)^2 + \Gamma_I^2)`

`b = (-2 \Gamma_I) / ((1 + \Gamma_R)^2 + \Gamma_I^2)`


Real part - Conductance

`g = (1 - \Gamma_R^2 - \Gamma_I^2) / ((1 + \Gamma_R)^2 + \Gamma_I^2)`

`g ((1 + \Gamma_R)^2 + \Gamma_I^2) = 1 - \Gamma_R^2 - \Gamma_I^2`

`g (1 + 2 \Gamma_R + \Gamma_R^2 + \Gamma_I^2) = 1 - \Gamma_R^2 - \Gamma_I^2`

`g + 2 g \Gamma_R + g \Gamma_R^2 + g \Gamma_I^2 = 1 - \Gamma_R^2 - \Gamma_I^2`

`g \Gamma_R^2 + \Gamma_R^2 + 2 g \Gamma_R + g \Gamma_I^2 + \Gamma_I^2 = 1 - g`

`\Gamma_R^2 (g + 1) + 2 g \Gamma_R + \Gamma_I^2 (g + 1) = 1 - g`

`\Gamma_R^2 + 2 g / (g + 1) \Gamma_R + \Gamma_I^2 = (1 - g) / (1 + g)`

`\Gamma_R^2 + 2 g / (g + 1) \Gamma_R + g^2 / (g + 1)^2 + \Gamma_I^2 - g^2 / (g + 1)^2 = (1 - g) / (1 + g)`

`(\Gamma_R + g / (g + 1))^2 + \Gamma_I^2 = (1 - g) / (1 + g) + g^2 / (1 + g)^2`

`(\Gamma_R + g / (g + 1))^2 + \Gamma_I^2 = ((1 - g)(1 + g)) / (1 + g)^2 + g^2 / (1 + g)^2`

`(\Gamma_R + g / (g + 1))^2 + \Gamma_I^2 = ((1 - g)(1 + g) + g^2) / (1 + g)^2`

`(\Gamma_R + g / (g + 1))^2 + \Gamma_I^2 = (1 + g - g - g^2 + g^2) / (1 + g)^2`

`(\Gamma_R + g / (g + 1))^2 + \Gamma_I^2 = (1 / (1 + g))^2`

Circle with center `[(-g) / (g + 1), 0]` and radius `1 / (1 + g)`.


Imaginary part - Susceptance

`b = (-2 \Gamma_I) / ((1 + \Gamma_R)^2 + \Gamma_I^2)`

`b ((1 + \Gamma_R)^2 + \Gamma_I^2) = -2 \Gamma_I`

`b (1 + 2 \Gamma_R + \Gamma_R^2) + b \Gamma_I^2 = -2 \Gamma_I`

`b + 2 b \Gamma_R + b \Gamma_R^2 + b \Gamma_I^2 = -2 \Gamma_I`

`2 b \Gamma_R + b \Gamma_R^2 + b \Gamma_I^2 + 2 \Gamma_I = -b`

`b (2 \Gamma_R + \Gamma_R^2) + b (\Gamma_I^2 + 2 \Gamma_I / b) = -b`

`b ((2 \Gamma_R + \Gamma_R^2) + (\Gamma_I^2 + 2 \Gamma_I / b)) = -b`

`\Gamma_R^2 + 2 \Gamma_R + 1 - 1 + \Gamma_I^2 + 2 1 / b \Gamma_I + 1 / b^2 - 1 / b^2 = -1`

`(\Gamma_R + 1)^2 + (\Gamma_I + 1 / b)^2 = -1 + 1 + 1 / b^2`

`(\Gamma_R + 1)^2 + (\Gamma_I + 1 / b)^2 = (1 / b)^2`

Circle with center `[-1, (-1) / b]` and radius `1 / b`.
